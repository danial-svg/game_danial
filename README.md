# بازی نجات قلعه

یک بازی دوبعدی ساخته‌شده با زبان برنامه‌نویسی Rust.

## ویژگی‌ها

* دارای چندین شخصیت مختلف
* سیستم پس‌زمینه (Background)
* مکانیک‌های مبارزه
* توسعه‌یافته با زبان Rust
* استفاده از تصاویر و فایل‌های گرافیکی

## ساختار پروژه

```text
assets/       فایل‌های گرافیکی و منابع بازی
src/          کدهای منبع پروژه
Cargo.toml    تنظیمات پروژه Rust
```

## اجرای بازی

برای اجرای بازی:

```bash
cargo run
```

## ساخت نسخه نهایی

برای ساخت فایل اجرایی:

```bash
cargo build --release
```

پس از اتمام ساخت، فایل اجرایی در مسیر زیر قرار می‌گیرد:

```text
target/release/
```
